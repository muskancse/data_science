 Python Data Manipulation & API Integration

A comprehensive Python project showcasing practical Data Analytics and Data Engineering skills using Pandas, REST APIs, and data transformation techniques.
Developed in VS Code and compatible with Google Colab and Jupyter Notebook environments.

This project demonstrates real-world workflows including dataset manipulation, API integration, relational data merging, and automation of Excel-like operations using Python.
Project Highlights
* Data Cleaning & Transformation using Pandas
* REST API Integration with OpenWeatherMap API
* Relational Data Merging using Primary & Foreign Keys
* CSV Import/Export Automation
* Statistical Analysis & Data Processing
* Excel Operations Replicated in Python
